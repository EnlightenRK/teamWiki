# 本Wiki项目的安装与使用

## 1.wiki安装(mkdocs安装)
_**预先环境：windows 、 Python2.7.8 +**_  
1.以管理员身份打开cmd 输入
```
pip install mkdocs
```
```
pip install mkdocs-material
```
2.安装完成  
## 2.本项目的导入
